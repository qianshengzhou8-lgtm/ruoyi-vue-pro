# 🏫 学校管理模块 - 前端使用指南

## 📋 概述

本系统是一个完整的学校用户管理平台，包含**管理后台**和**移动端**两部分。

## 🔑 登录方式

### 1. 管理后台登录（教职工使用）
- **访问地址**: `http://your-domain/admin/login`
- **用户类型**: 教职工（院长、校长、班主任、教师）
- **特点**: 
  - 功能全面，支持所有管理功能
  - 按角色显示不同菜单和数据权限
  - 支持导入导出功能

### 2. 移动端登录（学生使用）
- **访问地址**: `http://your-domain/app-api/member/auth/login`
- **用户类型**: 学生
- **特点**:
  - 简洁的选课界面
  - 只能查看和选择课程
  - 无法访问管理功能

## 🖥️ 管理后台功能入口

### 登录后如何进入学校管理模块

1. **登录系统**
   - 使用教职工账号（管理员创建）
   - 账号格式：`工号`，密码：`初始密码`

2. **进入学校管理**
   - 登录后会在左侧菜单看到"学校管理"菜单
   - 点击展开可以看到所有子菜单

## 🎯 角色权限说明

### 👨‍💼 校长（角色=3）
- **权限**: 全校所有数据
- **可见菜单**: 所有学校管理菜单
- **数据范围**: 可以查看全校所有学院、专业、班级、教职工、学生

### 👨‍🎓 学院院长（角色=2）
- **权限**: 本学院所有数据
- **可见菜单**: 学院管理、专业管理、班级管理、教职工管理、学生管理（部分）
- **数据范围**: 只能查看本院的学院、专业、班级、教职工、学生

### 👩‍🏫 班主任（角色=1）
- **权限**: 本班学生数据
- **可见菜单**: 班级管理、学生管理
- **数据范围**: 只能查看自己负责的班级和班级内的学生

### 👨‍🏫 教师（角色=0）
- **权限**: 基础查询
- **可见菜单**: 学院管理、专业管理、班级管理、教职工管理、学生管理（只读）
- **数据范围**: 只能查看列表，无法修改数据

## 📝 操作步骤

### 1. 学院管理
```
路径: /school/college
1. 点击"新增"按钮
2. 填写学院名称、排序、状态
3. 点击"确定"保存
```

### 2. 专业管理
```
路径: /school/major
1. 先创建学院（学院管理）
2. 点击"新增"按钮
3. 选择所属学院、填写专业名称
4. 点击"确定"保存
```

### 3. 班级管理
```
路径: /school/class
1. 先创建专业（专业管理）
2. 点击"新增"按钮
3. 选择所属专业、填写班级名称
4. 点击"确定"保存
```

### 4. 教职工管理
```
路径: /school/staff
1. 创建教职工账号
2. 选择所属学院
3. 设置角色（教师、班主任、院长、校长）
4. 设置负责的班级（仅班主任需要）
```

### 5. 学生管理
```
路径: /school/student
1. 创建学生账号
2. 选择所属班级
3. 系统自动创建移动端登录账号
```

### 6. 课程管理
```
路径: /school/course
1. 创建课程
2. 设置课程类型（必修/选修）
3. 设置授课教师
4. 选修课需设置容量限制
```

## 📊 导入导出功能

### 导入数据
1. 点击"导入"按钮
2. 下载模板文件
3. 填写数据后上传Excel
4. 系统自动导入并显示结果

### 导出数据
1. 设置筛选条件（可选）
2. 点击"导出"按钮
3. 自动下载Excel文件

## 📱 移动端功能

### 学生选课
```bash
# 1. 学生登录移动端
# 2. 进入"课程选课"页面
# 3. 查看可选课程列表
# 4. 点击"选课"按钮
# 5. 查看已选课程
```

### API端点
- **课程列表**: `GET /app/school/course/list`
- **选课**: `POST /app/school/course/{courseId}/select`
- **退课**: `POST /app/school/course/{courseId}/deselect`

## 🔍 数据权限测试

### 测试步骤
1. 使用不同角色登录系统
2. 观察左侧菜单差异
3. 查看数据列表的内容
4. 验证权限是否生效

### 预期效果
- **校长**: 看到全校所有数据
- **院长**: 只看到本院数据
- **班主任**: 只看到本班数据
- **教师**: 只看到基础查询，无法修改

## 💡 使用技巧

1. **级联选择**: 在创建学生时，会自动加载学院→专业→班级的级联数据
2. **快速搜索**: 每个页面都有搜索框，支持按名称搜索
3. **批量操作**: 支持批量导入数据，提高效率
4. **实时统计**: 课程管理页面有选课统计功能

## 🚀 快速开始

1. 运行SQL脚本创建菜单和角色
2. 使用校长账号登录系统
3. 按顺序创建：学院→专业→班级
4. 创建教职工和学生账号
5. 创建课程，开始选课流程
