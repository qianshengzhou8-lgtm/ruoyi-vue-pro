# 芋道（YuDao）- ruoyi-vue-pro 项目总结文档

## 一、项目概述

**芋道（YuDao）** 是一个基于 Spring Boot 多模块架构的中国国产开源快速开发平台，以开发者为中心，全部代码开源，个人与企业可 100% 免费使用。

项目基于 **RuoYi-Vue** 重构优化而来，在保留 RuoYi 原有系统功能基础上进行了大量代码重构（⭐️）和全新功能的扩展（🚀），覆盖了从基础系统管理到企业级业务系统的全栈能力。

- **开源协议**：MIT License（比 Apache 2.0 更宽松）
- **作者**：YunaiV（艿艿）
- **仓库地址**：
  - Gitee: https://gitee.com/zhijiantianya/ruoyi-vue-pro
  - GitHub: https://github.com/YunaiV/ruoyi-vue-pro
- **演示地址**：
  - Vue3 + element-plus: http://dashboard-vue3.yudao.iocoder.cn
  - Vue3 + vben(ant-design-vue): http://dashboard-vben.yudao.iocoder.cn
  - Vue2 + element-ui: http://dashboard.yudao.iocoder.cn

## 三、项目结构（Maven 多模块架构）

### 3.1 根模块

```
ruoyi-vue-pro-master/
├── yudao-dependencies/        # Maven BOM，统一依赖版本管理
├── yudao-framework/           # Java 框架拓展层（15个 Starter）
├── yudao-server/              # 主服务启动模块
├── yudao-module-*/            # 15个业务模块
├── yudao-ui/                  # 5个前端项目
├── sql/                       # 数据库脚本（含10种数据库支持）
├── script/                    # 部署和工具脚本
└── pom.xml                    # 根 POM
```

### 3.2 yudao-framework（框架层，15个模块）

| 模块 | 说明 |
|------|------|
| `yudao-common` | 通用工具类、枚举、异常、POJO、校验 |
| `yudao-spring-boot-starter-web` | Web 层封装（MVC、Swagger、统一返回等） |
| `yudao-spring-boot-starter-security` | 安全认证（Spring Security + Token + 多用户体系） |
| `yudao-spring-boot-starter-mybatis` | 持久层封装（MyBatis Plus + 多数据源） |
| `yudao-spring-boot-starter-redis` | 缓存封装（Redis + Redisson + Cache） |
| `yudao-spring-boot-starter-mq` | 消息队列封装（支持多种MQ实现） |
| `yudao-spring-boot-starter-job` | 定时任务封装（Quartz） |
| `yudao-spring-boot-starter-excel` | Excel 导入导出 |
| `yudao-spring-boot-starter-monitor` | 监控封装（Admin + Actuator + SkyWalking） |
| `yudao-spring-boot-starter-protection` | 服务保障（分布式锁、幂等、限流） |
| `yudao-spring-boot-starter-biz-tenant` | 多租户封装 |
| `yudao-spring-boot-starter-biz-data-permission` | 数据权限封装 |
| `yudao-spring-boot-starter-biz-ip` | IP 归属地解析 |
| `yudao-spring-boot-starter-websocket` | WebSocket 封装 |
| `yudao-spring-boot-starter-test` | 单元测试封装 |

## 四、核心功能特性

### 4.1 系统功能亮点
- **SaaS 多租户**：支持多租户体系，每个租户可自定义权限和套餐
- **多终端认证**：支持多种用户体系的认证系统，SSO 单点登录，OAuth2 授权
- **按钮级权限**：动态权限菜单，Redis 缓存提升性能
- **代码生成器**：一键生成 Java、Vue 前后端代码、SQL 脚本、接口文档，支持单表/树表/主子表
- **文件服务**：支持 S3（MinIO、阿里云、腾讯云、七牛云）、本地、FTP、数据库四种存储方式

### 4.2 工作流特性（企业级）
- **双设计器**：BPMN 标准设计器 + 仿钉钉/飞书 SIMPLE 设计器
- 会签、或签、依次审批（顺序会签）
- 驳回（任意节点）、转办、委派
- 加签（向前/向后）、减签
- 撤销、终止
- 超时审批、自动提醒（短信/邮件/站内信）
- 父子流程（同步/异步）
- 条件分支、并行分支、包容分支、路由分支
- 触发节点（HTTP请求/回调）、延迟节点

### 4.3 三方集成
- 微信小程序、微信公众号、企业微信、钉钉三方登录
- 支付宝、微信支付与退款
- 阿里云、腾讯云短信渠道
- 信创（国产化）适配：达梦、人大金仓、瀚高、神通等国产数据库

## 七、项目优势总结

1. **代码质量高**：按阿里巴巴 Java 开发手册规范编写，代码注释率高达 37%+
2. **架构整洁**：清晰的多模块 Maven 架构，框架层与业务层分离
3. **功能完善**：从基础管理到企业业务（OA/ERP/CRM/WMS/MES/CMS/商城/IM/AI/IoT），应有尽有
4. **国产化适配**：支持达梦、人大金仓、瀚高、神通等国产数据库，满足信创要求
5. **多端支持**：PC 端（Vue3/Vue2）、移动端（uni-app APP/H5/小程序）全平台覆盖
6. **完全开源**：MIT 协议，无商业版，代码 100% 公开，不做代码阉割
7. **开发效率高**：内置代码生成器，一键生成 CRUD；配置式功能灵活组合
8. **企业级特性**：SaaS 多租户、数据权限、工作流引擎、支付系统、分布式服务保障
